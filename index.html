Índice de la Guía

1. Subir a GitHub (Web) 2. Desplegar en Vercel 3. Conectar Google Sheets 4. Automatizar con Make

1. Subir tu archivo a GitHub desde el navegador

Este método te permite subir tu página web sin usar la terminal de comandos de tu sistema.

  1. Inicia sesión en tu cuenta en GitHub.com.
  2. Haz clic en New (Nuevo) para crear un nuevo repositorio.
  3. Nómbralo memoriaviva-ai y déjalo como Público. No inicialices ningún archivo.
  4. En la pantalla que aparece, haz clic en el enlace que dice create a new file (crear un nuevo archivo).
  5. Escribe exactamente index.html en el nombre del archivo.
  6. Copia el código de esta página e insértalo en el editor.
  7. Haz clic en Commit changes... abajo en el botón verde y confírmalo para publicar.

2. Despliegue instantáneo en Vercel

Vercel tomará automáticamente el archivo index.html de tu GitHub y lo publicará con un link público seguro (HTTPS).

  1. Entra a Vercel.com e inicia sesión usando tu cuenta de GitHub.
  2. Haz clic en Add New... > Project en tu panel de control de Vercel.
  3. Importa el repositorio de GitHub llamado memoriaviva-ai que acabas de crear.
  4. Deja todas las opciones por defecto y haz clic en el botón Deploy.
  5. ¡Y listo! Vercel te entregará una dirección web en vivo para compartir con tus usuarios.

3. Guardar datos en Google Sheets (Sin servidor)

Puedes almacenar la información del formulario directamente en una hoja de Google Drive gratis:

  1. Crea una hoja en Google Drive llamada Leads MemoriaViva.
  2. Agrega cuatro columnas en la primera fila: A1: Fecha, B1: Nombre, C1: Correo, D1: Mensaje.
  3. En el menú superior, ve a Extensiones > Apps Script.
  4. Pega el código de abajo en el editor y haz clic en Guardar (ícono de disquete):
function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    
    // Inserta nueva fila con fecha actual, nombre, correo y mensaje del formulario
    sheet.appendRow([new Date(), data.name, data.email, data.message]);
    
    return ContentService.createTextOutput(JSON.stringify({"result":"success"}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({"result":"error", "error": error.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

4. Automatización de correos de bienvenida con Make

Si quieres automatizar tus correos electrónicos o enviar notificaciones instantáneas a tu equipo:

  1. Ingresa a Make.com y crea un nuevo escenario.
  2. Agrega el disparador principal buscando Webhooks > Custom Webhook.
  3. Crea un webhook, dale un nombre y copia la URL entregada (Ej. https://hook.us1.make.com/...).
  4. Pega esa URL en la configuración de la landing page o en la constante del archivo.
  5. En Make, presiona el botón Redetect y envía el formulario de prueba desde la landing page para vincular los datos.
  6. Agrega el siguiente módulo en Make (ej. Gmail, Mailchimp o Google Sheets) para realizar la acción automatizada deseada. ¡Listo!
